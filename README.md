## How to run 

1. Clone this project
```shell
$git clone https://github.com/joong8812/deepbrain-my-client.git
```

2. Install packages which it uses
```shell
$npm install
```

3. Run (default port 3000)
```shell
$yarn dev
```
or
```shell
$npm run dev
```
